# simple-blog

> 这是一个基于Vue+Element+Express的简易博客，在一般博客的基础上增加了很少的功能，减少了很多的功能

# 启动方式

> 0. 配置好数据库

* 本项目使用Mysql数据库进行存储
* 请将blogData数据文件内的blog_system文件夹放置于mysql目录下的data

> 1. 项目配置

```js
yarn install
// 或者
npm install
```

> 2. 启动服务器

```js
yarn run kokoro
// 或者
npm run kokoro
```

> 3. 启动客户端

```js
yarn run kokoro
// 或者
npm run kokoro
```

> 4. 打包

```js
yarn run build
// 或者
npm run build
```
